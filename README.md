# English Website

## Live website
[View the live site here](https://badralioui.github.io/css-html/)

## Project Overview

The purpose of this project was to demonstrate my understanding of the basic foundations of **HTML** and **CSS**.

This website was built as a simple front-end practice project to show that I can structure a webpage correctly with HTML and style it with CSS. The main goal was not to create a complex application, but to prove that I understand how to build and organise a web page using the core concepts of front-end development.

## Objective

The objective of this project was to show knowledge of:

- HTML page structure
- CSS styling
- Layout organisation
- Navigation design
- Sidebar sections
- Hover effects
- Colours, gradients, and typography
- Basic responsive design with media queries

## Technologies Used

- **HTML5**
- **CSS3**
- **Font Awesome** for icons

## Project Structure

The page includes:

- A **header** with the website title and a search box
- A **navigation menu**
- An **introduction section**
- A **left sidebar** with lesson links
- A **right sidebar** with key benefits
- A **footer**

## What I Practised

Through this project, I practised several important HTML and CSS concepts:

### HTML
- Creating a clear page structure
- Using headings, paragraphs, links, lists, forms, and divisions
- Organising content into logical sections

### CSS
- Styling text, backgrounds, and buttons
- Using **gradients**
- Positioning elements with **float**, **relative positioning**, and **transform**
- Creating hover effects for links
- Designing sidebars and navigation areas
- Applying spacing with padding and margins
- Using a **media query** for basic responsiveness on larger screens

## Design Choices

I used soft colours, gradients, and simple typography to create a friendly and educational look.  
The layout was designed to look like an English learning website, with clear sections that are easy to read and navigate.

## Responsive Design

A basic media query was added to slightly adjust the position of some elements on larger screens. This shows an understanding of responsive design principles and how layouts can be adapted depending on screen size.

## Learning Outcome

This project helped me strengthen my knowledge of the fundamentals of front-end development.  
It shows that I understand how HTML and CSS work together to build the structure and presentation of a website.

Even though the project is simple, it reflects the essential skills needed to create a clean and organised static webpage.

## Future Improvements

In the future, this project could be improved by:
